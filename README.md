# 연산 타임어택 (Math Time Attack)

구구단부터 99단까지! 곱셈 실력을 테스트하는 타임어택 게임

## 앱 소개

**연산 타임어택**은 빠르고 정확한 곱셈 계산 능력을 키울 수 있는 교육용 게임입니다.
3가지 난이도로 자신의 실력에 맞게 도전할 수 있으며, 최고 기록을 갱신하는 재미를 느낄 수 있습니다.

### 주요 기능

- **3단계 난이도**: 초급(구구단), 중급(19단), 고급(99단)
- **타임어택 모드**: 5문제를 최대한 빠르게 풀어보세요
- **최고 기록 저장**: 난이도별 최고 기록이 자동 저장됩니다
- **즉각적인 피드백**: 정답/오답 즉시 확인 가능
- **접근성 지원**: 스크린 리더 호환, ARIA 속성 적용

### 난이도별 설명

| 난이도 | 범위 | 대상 |
|--------|------|------|
| 초급 | 1~9단 (구구단) | 초등학생, 입문자 |
| 중급 | 1~19단 | 중학생, 중급자 |
| 고급 | 1~99단 | 고급자, 도전자 |

## 플레이 방법

1. **난이도 선택**: 초급/중급/고급 중 선택
2. **문제 풀기**: 화면에 표시된 곱셈 문제의 답을 입력
3. **정답 제출**: 확인 버튼 또는 Enter 키로 제출
4. **기록 확인**: 5문제 완료 후 소요 시간과 최고 기록 확인
